### SHA-512
* The algorithm takes as input a message with < $2^{128}$ bits and produce a $512$-bit digest. The block length is 1024-bit
* The steps:
	1. Apply padding bits so that $length\_of\_message \equiv 896\ (mod\ 1024)$
	2. Apply 128 bits that is treated as the length of the message (so now $896 + 128 = 1024$)
	3. A 512-bit buffer(8 64-bit registers) is used to hold the intermediate and results of the hash.

5.  Process the image in 1024-bit blocks, using a module $F$ that consists of 80 rounds. The first round takes the initial input $H_{i-1}$. Each round $t$ make use of $W_t$ derived from the message block $M_i$ and $K_t$. The last round is added to the first via addition mod $2^{64}$
> <span style="color:3bba10">Question: Why is there this add step?</span>
#### The round function
- $T_{1} = h + \mathrm{Ch}(e, f, g) + (\sum_{1}^{512} e) + W_{t} + K_{t}$
- $T_{2} = (\sum_{0}^{512} a) + \mathrm{Maj}(a, b, c)$
- $h = g$
- $g = f$
- $f = e$
- $e = d + T_{1}$
- $d = c$
- $c = b$
- $b = a$
* $a = T_{1} + T_{2}$
		
Where:
* $t = \mathrm{step\ number};\ 0 \leq t \leq 79$
* $\mathrm{Ch}(e, f, g) = (e\ \mathrm{AND}\ f) \oplus ((\mathrm{NOT}\ e\ \mathrm{AND}\ g))$ -- if $e$ then $f$ else $g$
* $\mathrm{Maj}(a, b, c) = (a\ \mathrm{AND}\ b) \oplus (a\ \mathrm{AND}\ c) \oplus (b\ \mathrm{AND}\ c)$ -- == 1 only if 2 or 3 arguments are **true
* $(\Sigma_0^{512} a) = \mathrm{ROTR}^{28}(a) \oplus \mathrm{ROTR}^{34}(a) \oplus \mathrm{ROTR}^{39}(a)$
* $(\Sigma_1^{512} e) = \mathrm{ROTR}^{14}(e) \oplus \mathrm{ROTR}^{18}(e) \oplus \mathrm{ROTR}^{41}(e)$
* $\mathrm{ROTR}^n(x) = \mathrm{circular\ right\ shift\ (rotation)\ of\ the\ 64\text{-}bit\ argument\ x\ by\ } n \mathrm{\ bits}$
* $W_t$ - a word derived from the current 1024-bit block
* $K_t$ - a word constant
* $+$ - addition mod $2^{64}$

$W_t$ is generated by:
		$W_{t} = \sigma_{1}^{512}(W_{t-2}) + W_{t-7} + \sigma_{0}^{512}(W_{t-15}) + W_{t-16}$
Where:
* $\sigma_{0}^{512}(x) = \mathrm{ROTR}^{1}(x) \oplus \mathrm{ROTR}^{8}(x) \oplus \mathrm{SHR}^{7}(x)$
* $\sigma_{1}^{512}(x) = \mathrm{ROTR}^{19}(x) \oplus \mathrm{ROTR}^{61}(x) \oplus \mathrm{SHR}^{6}(x)$
